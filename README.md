# 运行参数

```
-debug 打开调试模式，beego设置为DEV模式，静态资源调用statik，关闭时，beego设置为PROD模式，静态资源使用web/static路径资源。默认：false

-mode 运行模式：web：Web模式；默认：web

-ip 应用监听IP地址。默认：127.0.0.1

-port 应用监听端口。默认：随机
```
